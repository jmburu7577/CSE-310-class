│ 🚀 GETTING STARTED                                                        │
└───────────────────────────────────────────────────────────────────────────┘

STEP 1: Create First Account (Becomes Admin)
  1. Open http://localhost:5000
  2. Click "Sign Up"
  3. Fill in your details
  4. ✅ You're now an Administrator!

STEP 2: Create Additional Users
  1. Have others sign up (they become Viewers)
  2. Or create test accounts yourself

STEP 3: Assign Roles (Admin Only)
  1. Login as Admin
  2. Click "Manage Users" button
  3. View all users
  4. Change roles as needed
  5. ✅ Changes take effect immediately!

ROLE MANAGEMENT:

  GET  /api/auth/roles         - Get available roles

PROTECTED EMPLOYEE ENDPOINTS:
  GET    /api/employees        - Requires: Employees.View
  POST   /api/employees        - Requires: Employees.Add
  PUT    /api/employees/{id}   - Requires: Employees.Edit
  DELETE /api/employees/{id}   - Requires: Employees.Delete
  GET    /api/employees/stats  - Requires: Statistics.View

┌───────────────────────────────────────────────────────────────────────────┐
│ ⚠️ COMMON ISSUES                                                          │
└───────────────────────────────────────────────────────────────────────────┘

ISSUE: Can't see Add Employee button
  → Your role doesn't have permission
  → Contact Admin to upgrade your role

ISSUE: Getting 403 Forbidden errors
  → You don't have permission for that action
  → Check your role in the header

ISSUE: Can't change user roles
  → Only Administrators can change roles
  → You need Admin access

ISSUE: First user isn't Admin
  → Delete users.json and create new first user
  → First user is always Admin
