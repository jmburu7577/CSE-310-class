🔧 TROUBLESHOOTING:

Problem: "dotnet: command not found"
Solution: Install .NET SDK from https://dotnet.microsoft.com/download

Problem: Port 5000 already in use
Solution: Run this first: Get-Process dotnet | Stop-Process -Force

Problem: Browser shows "Can't reach this page"
Solution: Make sure you see "Now listening on: http://localhost:5000" in PowerShell

Problem: Blank page in browser
Solution: Press Ctrl+Shift+R to hard refresh the page
